MATLAB toolbox to process SH2/PTB Domain microarray images and perform statistical analysis
Implemented by Greg Koytiger (koytiger@fas.harvard.edu)

Code is presented as is, without any warranty or support.

Unzip into a directory of your choosing. Analysis done using MasterScript.m

Code is able to perform:

-Robust fitting of 96 well SH2/PTB Domain microarray images into saturation binding curves
-Filters titrations by R^2 and fold over background
-Merges replicate data points
-Performs statistical tests on enrichment among oncogenes
-Generates figures, generates inputs for Circos plots
